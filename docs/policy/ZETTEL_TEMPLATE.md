# Zettel Template

Every promoted note must follow this structure to ensure long-term graph health.

```markdown
---
tags: [layer/wiki, status/evergreen]
topics: []
---

# Proposed Title

## Core Idea
(1-2 sentences summarizing the unique insight)

## Source / Spark
- Link to original Resource or Project note.

## Related
- [[Link to existing concept]]

## Follow-up Thoughts
(Questions or extensions for future thinking)
```
